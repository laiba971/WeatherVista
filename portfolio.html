About Me

Hi! I'm Laiba Ijaz, and I built a weather forecasting website using HTML, CSS, and JavaScript. This project shows real-time weather info using a weather API.

🌦️ WeatherVista

This project is a responsive website that provides real-time weather updates and 5-day forecasts using the OpenWeatherMap API. Users can search for any city, view current weather conditions (like temperature, humidity, wind), and register or log in to access their dashboard.

  • 🌍 Location-based weather search
  • 📅 5-day forecast with intuitive UI
  • 🔒 Authentication: Signup, Login, and Forgot Password
  • 📱 Fully responsive and friendly
